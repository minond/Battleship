Game instructions:

Select your armada and strategically place them across the board to ensure they are safe from your oppenent's attacks.

Once you and your opponent are ready, the battle may begin. An attack if fired By clicking on a panel on your opponent's board. If you hit one of your opponent's ships you will be notified of the successful attack, otherwise, a marker will be placed on the location as a reminder.

Once a ship has been hit on all of its points, it sinks. The first player to sink all of his or her opponent's ships will be declared the winner of the battle.

Preliminary Software Design:
  • Ship model - represents an available ship piece.
    • string name - the ship's name (Clipper, Pirate ship, etc.)
    • int length
    • int width
    • string src - a URL to the ship's image.
  • Piece model - represent a ship on the board.
    • Ship ship - a reference to a Ship model (ship piece represents).
    • Board board - a reference to a Board model.
    • int[] coordinates - piece's coordinates on the board.
    • boolean alive() - returns true of piece has not been sunk.
    • void attacked_on(int x, int y) - hit a panel.
    • boolean falls_within(int x, int y) - returns true if a piece falls within the specified coordinates.
  • Board model - a game board.
    • string player_name
    • Piece[] pieces - collection of ship pieces on board.
    • boolean valid_location(int x, int y, Ship ship) - return true if we can place a piece on the board at the specified location.
    • boolean trigger_attack(int x, int y) - fire an attack at a specific coordinate.
    • boolean has_live_pieces() - returns true of board has pieces that haven't been hit.
    • boolean has_live_piece_at(int x, int y) - returns true if a piece falls within the specified coordinates (see Piece.falls_within).
